Pricing overview

Google Maps uses a pay-as-you-go model. You get a free monthly usage quota, and any additional usage is charged based on the number of requests made.

Free usage caps

Many SKUs include a monthly free usage cap. This example shows how the first billable events can be $0.

Usage range Billable events Price Calculation
0 - 10,000 free cap First 10,000 events $0 $0 * 10,000 / 1,000 = $0.00
Free usage caps are applied per SKU before billed usage is calculated.

Pricing example

If an application sends 10,000 Autocomplete Requests in a month, those requests are covered by the free cap and are not billed.

Example: Dynamic Maps 100,000 events
10,000 free events
90,000 billable events at $7.00 per 1,000
Total month cost = (90,000 / 1000 ) * 7.0 = $630.00
